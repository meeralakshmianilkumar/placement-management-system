# Placement Management System

A web-based Placement Management System developed to streamline campus recruitment activities for students, placement officers, and recruiters.

## Features

### Student Module
- Student registration and login
- View available placement drives
- Apply for placement opportunities
- Track application status
- View eligibility criteria

### Placement Officer Module
- Secure officer login
- Add and manage placement drives
- View registered students
- Verify student eligibility
- Manage applications
- Monitor placement activities

### Company/Drive Management
- Add company details
- Create placement drives
- Set eligibility criteria
- View applicants for each drive

## Technologies Used

- Python
- Flask
- HTML
- CSS
- Jinja2 Templates
- SQLite Database

## Installation

1. Clone the repository

```bash
git clone https://github.com/USERNAME/placement-management-system.git
```

2. Navigate to the project directory

```bash
cd placement-management-system
```

3. Install dependencies

```bash
pip install flask
```

4. Run the application

```bash
python app.py
```

5. Open your browser and visit

```
http://127.0.0.1:5000
```
